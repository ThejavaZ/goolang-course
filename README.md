# 🐹 Go Course: De Cero a Backend

¡Bienvenido! Este es mi repositorio personal de aprendizaje de **Go (Golang)**. El objetivo de este curso es migrar mis conocimientos de **.NET (C#)** y **Laravel (PHP)** hacia la eficiencia y concurrencia de Go, enfocado en el desarrollo de sistemas robustos y microservicios.

> [!IMPORTANT]
> **Colaboración IA:** Este curso ha sido estructurado y desarrollado en colaboración con **Gemini**, actuando como tutor técnico para asegurar las mejores prácticas de la industria y una transición fluida entre lenguajes.

## 🚀 Estructura del Proyecto

El curso está organizado en módulos incrementales, diseñados para ser consultados como una referencia rápida en el futuro:

- **00-introduccion/**: Configuración del entorno en Fedora Linux, `go mod` y estructura de proyectos.
- **01-fundamentos/**: Tipado estático, casting explícito y variables.
- **02-estructuras-datos/**: El corazón de Go: Slices (dinámicos) y Maps (diccionarios).
- **03-funciones-metodos/**: Retornos múltiples y comportamiento sobre Structs.
- **04-interfaces/**: Composición sobre herencia y polimorfismo implícito.
- **05-manejo-errores/**: La filosofía de "los errores son valores" (no más try-catch).
- **06-concurrencia/**: Goroutines, WaitGroups y Channels.
- **07-libreria-estandar/**: Servidores HTTP nativos y manejo de JSON.
- **08-ejercicios/**: Retos de algoritmos para fortalecer la lógica.
- **09-proyecto-final/**: (En desarrollo) Sistema de gestión funcional.

## 🛠️ Stack de Desarrollo

- **OS**: Fedora Linux 43 (KDE Plasma)
- **Lenguaje**: Go 1.22+
- **Editor**: VS Code / Vim
- **Identidad**: "Generación de la Frontera" (Mezcla de cultura técnica del norte de México y estándares globales).

## 📖 Cómo usar este repo

Cada carpeta contiene archivos `.go` comentados que explican el "por qué" de cada sintaxis. Para ejecutar cualquier módulo:

```bash
cd nombre-del-modulo
go run archivo.go
```
